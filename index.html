WOD Champ | The Story

WOD Champ was conceived in the summer of 2024 as a senior project and has since evolved into a feature complete CrossFit web application. It is designed to simplify gym management for coaches and enhance participation for members. For coaches it provides a user-friendly platform for tasks like user management and workout scheduling. For gym members, WOD Champ offers an easy way to sign up for workouts, track progress, and access instructional videos on various CrossFit movements. Grounded in a Christian worldview, WOD Champ promotes physical well-being, community, and fellowship while upholding Biblical values, making it a meaningful tool for not just physical fitness but also mental and spiritual growth.

Technical Details

Framework

WOD Champ is built on Microsoft's latest ASP.NET web framework, Blazor. The project is configured for Blazor Server hosting, enabling efficient real-time interactions between the client and server via SignalR, a free and open-source library and protocol developed by Microsoft.

The project follows an N-tier design pattern with multiple layers. The first is the Data Access Layer (DAL), managed by Entity Framework and its DbContext. Above that is the Business Logic Layer (BLL), which consists of both custom and pre-built services. Finally, the Presentation Layer comprises the user interface (UI), built with various Razor components, delivering a seamless, responsive experience while allowing easy modification and code reuse.

Database

Microsoft SQL Server (MSSQL) was selected as the database backend of choice for WOD Champ due to its robust performance, scalability, and seamless integration with the .NET ecosystem. SQL Server provides strong support for Entity Framework, which simplifies data access through its powerful object-relational mapping (ORM) capabilities. Additionally, it offers advanced features such as security, data encryption, and transaction management, ensuring that user data is handled securely and efficiently. With SQL Server's high availability and cloud-readiness, the application can scale to meet future demands while maintaining optimal performance.

Hosting

Similar to the reasons for choosing Microsoft SQL Server, Microsoft Azure was selected as the cloud provider for WODChamp.com due to its seamless integration with the Microsoft ecosystem. Azure offers a robust, scalable, and flexible hosting environment, making it an ideal choice for Blazor Server applications like this one. With Azure, the application benefits from real-time interactions, low-latency performance, and high availability through its global network of data centers. Publishing is also made easy with Microsoft's built-in Visual Studio publishing profiles.

Code Example

The WOD Champ Blazor web app makes use of dependency injection (DI) to manage the lifecycles and dependencies of various services used throughout the application. By registering services at startup, services can be injected into Blazor components, controllers, or Razor pages where needed, allowing for greater de-coupled code and improving maintainability.
